# B. Getting Started

### Front :

| I     | SOP                                                                          |
| ----- | ---------------------------------------------------------------------------- |
| dev   | `yarn` -> `yarn start`                                                       |
| build | `yarn build` & revise `.env`, changing `REACT_APP_URL` to your server domain |

### Back :

| I     | SOP                                                                               |
| ----- | --------------------------------------------------------------------------------- |
| dev   | `yarn` -> `yarn watch`                                                            |
| build | revise your `.env`, changing `OPENAI_API_KEY` & `MONGO_URL` to your server domain |
